Sockets

Sockets

  • socket_accept - 接受套接字上的连接
  • socket_addrinfo_bind - 从给定的主机名创建并绑定到套接字
  • socket_addrinfo_connect - 从给定的主机名创建并连接到套接字
  • socket_addrinfo_explain - 获取有关的主机名的信息
  • socket_addrinfo_lookup - 获取包含有关给定主机名的内容的数组
  • socket_bind - 给套接字绑定名字
  • socket_clear_error - 清除套接字或者最后的错误代码上的错误
  • socket_close - 关闭套接字资源
  • socket_connect - 开启一个套接字连接
  • socket_create_listen - 打开端口上的套接字以接受连接
  • socket_create_pair - 创建一对不可区分的套接字，并将它们存储在数组中
  • socket_create - 创建一个套接字（通讯节点）
  • socket_export_stream - 将套接字扩展资源导出到封装套接字的流中
  • socket_get_option - 获取套接字的套接字选项
  • socket_getopt - 获取套接字的套接字选项
  • socket_getpeername - 查询给定套接字的远程端
  • socket_getsockname - 查询给定套接字的本地端
  • socket_import_stream - 导入流
  • socket_last_error - 返回套接字上的最后一个错误
  • socket_listen - 侦听套接字上的连接
  • socket_read - 从套接字读取最大长度的字节
  • socket_recv - 从已连接的套接字接收数据
  • socket_recvfrom - 从套接字接收数据，无论它是否连接
  • socket_select - 在给定的套接字数组上以指定的超时时间运行 select 系统调用
  • socket_send - 将数据发送到连接的套接字
  • socket_sendto - 向套接字发送消息，无论是否已连接
  • socket_set_block - 在套接字资源上设置阻塞模式
  • socket_set_nonblock - 在套接字资源上设置非阻塞模式
  • socket_set_option - 设置套接字的套接字选项
  • socket_setopt - 设置套接字的套接字选项
  • socket_shutdown - 关闭用于接收，发送或同时接收和接收的套接字
  • socket_strerror - 返回描述套接字错误的字符串
  • socket_write - 写入套接字
  • socket_wsaprotocol_info_export - 导出 WSAPROTOCOL_INFO 结构
  • socket_wsaprotocol_info_import - 从另一个进程导入套接字
  • socket_wsaprotocol_info_release - 释放导出的 WSAPROTOCOL_INFO 结构

socket_accept

<?php

$socket = socket_create(AF_INET, SOCK_STREAM, SOL_TCP);
socket_bind($socket, '127.0.0.1', 8080);
socket_listen($socket);
while (true) {
    var_dump(socket_accept($socket));
}

    }
}

socket_recv

<?php

$socket = socket_create(AF_INET, SOCK_STREAM, SOL_TCP);
socket_bind($socket, '127.0.0.1', 8080);
socket_listen($socket);
while (true) {
    if (($resource = socket_accept($socket)) !== false) {
        var_dump(socket_recv($resource, $buf, 1024, MSG_OOB));
        var_dump($buf);
    }
}

socket_recvfrom

<?php

$socket = socket_create(AF_INET, SOCK_STREAM, SOL_TCP);
socket_bind($socket, '127.0.0.1', 8080);
socket_listen($socket);
while (true) {
    if (($resource = socket_accept($socket)) !== false) {
        var_dump(socket_recvfrom($resource, $buf, 1024, MSG_OOB, $name, $port));
        var_dump($buf, $name, $port);
    }
}
