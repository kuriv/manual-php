# phar://

|           属性            | 支持 |
| :-----------------------: | :--: |
| 受 `allow_url_fopen` 限制 |  No  |
| 受 `allow_url_include` 限制 |  No  |
|         允许读取          | Yes  |
|         允许写入          | Yes  |
|         允许添加          |  No  |
|      允许同时读和写       |  Yes  |
|       支持 `stat()`       |  Yes  |
|      支持 `unlink()`      |  Yes  |
|      支持 `rename()`      |  Yes  |
|      支持 `mkdir()`       |  Yes  |
|      支持 `rmdir()`       |  Yes  |
