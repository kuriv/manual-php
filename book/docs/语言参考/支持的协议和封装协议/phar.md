# phar://

|           属性            | 支持 |
| :-----------------------: | :--: |
| 受 `allow_url_fopen` 限制 |  No  |
|         允许读取          |  No  |
|         允许写入          | Yes  |
|         允许添加          | Yes  |
|      允许同时读和写       |  No  |
|       支持 `stat()`       | Yes  |
|      支持 `unlink()`      | Yes  |
|      支持 `rename()`      | Yes  |
|      支持 `mkdir()`       | Yes  |
|      支持 `rmdir()`       | Yes  |
