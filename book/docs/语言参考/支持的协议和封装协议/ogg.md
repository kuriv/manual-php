# ogg://

通过 `ogg://` 封装协议读取的文件，是作为 OGG / Vorbis 格式的压缩音频编码，同样，通过 `ogg://` 封装协议写入或追加的数据格式也是压缩音频。 

|           属性            | 支持 |
| :-----------------------: | :--: |
| 受 `allow_url_fopen` 限制 |  No  |
|         允许读取          | Yes  |
|         允许写入          | Yes  |
|         允许添加          | Yes  |
|      允许同时读和写       |  No  |
|       支持 `stat()`       |  No  |
|      支持 `unlink()`      |  No  |
|      支持 `rename()`      |  No  |
|      支持 `mkdir()`       |  No  |
|      支持 `rmdir()`       |  No  |
